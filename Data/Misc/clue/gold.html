The ClueWeb09 Dataset

The ClueWeb09 dataset was created to support research on information retrieval and related human language technologies. It consists of about 1 billion web pages in ten languages that were collected in January and February 2009. The dataset is used by several tracks of the TREC conference.

Dataset Specifications

1,040,809,705 web pages, in 10 languages

Dataset Distribution:

The ClueWeb09 dataset and subsets are distributed in several different ways.

The full dataset is distributed as tarred/gzipped files on four 1.5 terabyte (TB) hard disk drives, in Linux ext3 format. The physical drives are standard SATA 3 Gbit/sec (SATA/300) 3.5" drives that should be compatible with any SATA/300 interface, including external USB to SATA/300 enclosures.

Full, 4 x 1.5TB:

The full dataset is also available as tarred/gzipped files on two 3.0 terabyte (TB) hard disk drives, in Linux ext3 format . The physical drives are standard SATA 6 Gbit/sec (SATA/600) 3.5"

Full, 2 x 3.0TB:

Check that your system’s operating system and hardware are compatiable with 3 TB drives (not all SATA external enclosures are compatiable with these drives).

The TREC "Category B" subset of the full dataset is distributed as tarred/gzipped files on one 1.0 gigabyte (GB) hard disk drive, in Linux ext3 format. The physical drive is a SATA 3 Gbit/sec (SATA/300) 3.5" drives that should be compatible with any SATA/ interface, including external USB to SATA 300 enclosures.

CatB, 1 x 500GB:

The Japanese subset of the full dataset is distributed as tarred/gzipped files on one 500 gigabyte (GB) hard disk drive, in Linux ext3 format. The physical drive is a SATA 3 Gbit/sec (SATA/300) 3.5" drive that should be compatible with any SATA/300 interface, including external USB to SATA/300 enclosures.

JA, 1 x 500GB:

File Formats and Sample Data:

Web pages are in the WARC file format. Each WARC file is about 1 gigabyte, uncompressed. Each WARC file contains several tens of thousands of web pages (e.g., 40,000). Each WARC file is compressed by gzip.

Online Services

The Lemur Project provides several online services to simplify use of the ClueWeb09 dataset.

Batch Query Service for ClueWeb09: Use the to search the Category A English or Category B dataset

Category A - English Interactive Search: Use the Indri search engine to interactively search the English part of the Category A dataset

Category B Interactive Search: Use the Indri search engine to interactively search the Category B dataset

Wikipedia Interactive Search: Use the Indri search engine to interactively search the Wikipedia part of the dataset

ClueWeb09 Attribute Lookup Service: Fast lookup of ClueWeb09 document attributes.
